# croparray

**Filetype:** _MATLAB&reg; function_

**Synopsis:** _returns cropped/central portion of 3D array_

Syntax
    B = croparray( A, desiredSize )
    
    returns central portion of 3D array A: data outside 'desiredSize' is
    trimmed away.

see also PADARRAY( )
